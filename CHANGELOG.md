# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-07-06

### Added
- **Clock Module**: Live digital clock with 12/24-hour format, date, and day display
- **Timer Module**: Complete timer with start, pause, resume, stop, reset controls
- **Stopwatch Module**: Millisecond-precision stopwatch with lap timing
- **Floating Overlay Widget**: Draggable floating timer that works above all apps
- **Foreground Service**: Background countdown with notification controls
- **Settings Screen**: Theme selection, alarm sound choice, vibration toggle, time format preference
- **Timer Presets**: 5, 10, 15, 25, 30, 45, 60 minute presets with custom preset support
- **Alarm System**: Premium "Time's Up" screen with alarm sound, vibration, and action buttons
- **Material Design 3**: Modern UI with smooth animations and responsive layouts
- **MVVM Architecture**: Clean code with dependency injection using Hilt
- **Room Database**: Persistent storage for timer presets and settings
- **Kotlin Coroutines**: Efficient async operations with StateFlow
- **GitHub Actions**: CI/CD workflows for automated builds and quality checks
- **Comprehensive Tests**: Unit and instrumented tests for core functionality

### Features
